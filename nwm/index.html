Abstract

Navigation is a fundamental skill for agents with visual-motor capabilities. We propose a Navigation World Model (NWM), a controllable video generation model that predicts future visual observations based on past observations and navigation actions. NWM, a Conditional Diffusion Transformer (CDiT) with 1B parameters, is trained on diverse video data from human and robot agents, including unlabeled egocentric video. The model scales favorably across known and unknown environments and exhibits improved navigation planning skills, either by planning from scratch or ranking proposals from an external policy. Unlike supervised models, NWM adapts to new constraints and leverages learned visual priors to imagine navigation trajectories from a single input image.

Planning with NWM + NoMaD

We include planning examples using NWM with an external navigation policy (NoMaD). Trajectories from NoMaD are ranked by NWM, which generates trajectory videos and selects the highest-scoring one.

Limitations

One failure mode in unknown environments is mode collapse, where the model generates frames similar to the training data.
